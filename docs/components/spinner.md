# Spinner / Loader

Overview
- Small inline or block spinner used to indicate loading states.

HTML
```html
<div class="ff-spinner" role="status" aria-label="Loading"></div>
```

Classes
- `.ff-spinner` — base spinner
- `.ff-spinner-sm`, `.ff-spinner-lg` — size modifiers

Accessibility
- Use `role="status"` and `aria-label` or visible text for screen readers.
